Javascript Conditionals

    if = takes a single expression inside of a parathenisis which it will evaluate to be true or false

    elseif = follows an If statement, it is secondary condition, works exactly like an If statement

    else = the "otherwise" section. Does not take an expression in parathenisis
// If age is negative if(age < 0){ console.log("Come back once you're out of the womb"); } // If age is 21 if(age === 21){ console.log("Happy 21st Birthday!"); } // If age is odd //(not evenly divisible by two) if(age % 2 !== 0) { console.log("Your age is odd!"); }
