While Loops

A while loop repeats the code WHILE the condition is true

                while(someCondition){
                    //run some code
                }

A good example of that would be:

                var count = 1;
                while(count < 6) {
                    console.log("count is: " + count);
                    
                    count++;
                }

Now to run the exact code as above BUT add two to it each time:

                var count = 1;
                while(count < 6) {
                    console.log("count is: " + count);
                    count+=2;
                }

Another while loop example, this time printing each letter of the work "hello"


                
                var str = "hello";
                var count = 0;

                while(count < str.length) {
                    console.log(str[count]);
                    count++;
                }

                Remember: the length is always 1 great than the highet index of a string.
